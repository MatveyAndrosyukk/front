1.Cкачать картинку фавиконки в svg через ctlr + g, перейти на сайт real favicon generator, все сгенерировать, настроить, ссылки полученные вставить в html,
содержимое архива вставить в /source/icons и поменять путь к картинкам в ссылках html, browserconfig, site.webmanifest и там же поставить
'"background_color": "transparent"'. файлы browserconfig и site.webmanifet перенести на уровень к index.html.

2.Cкопировать нужную картинку в svg, создать svg-файл и туда все вставить, затем изменить в теге svg fill="currentCollor", везде где fill отвечет
за цвет самой svg, полностью стереть его, добавить к тегу svg id="logo", если есть заливка, в основном она белая, то поменять fill="white" на
fill="transparent", вставить svg в html так: <svg class="header__logo" viewBox="0 0 199 44" role="img" aria-label="Логотип YourMeal">
-> <use href="source/img/logo.svg#logo"/>.

3.Открыть сайт IconMoon, нажать на 3 линии, выбрать import to set, выбрать все сохраненные иконки, удалить цвета, если иконка
скачалась плохо, то нужно ее дселать отдельно. выделить иконки, нажать generate fonts -> нажать на шестеренку, в модалке указать font name - "icons",
class prefix - "_icon-", создать папку fonts и туда сохранить иконки. далее извлечь в текущую папку. нужен только файл icons.ttf, вынести его наверх и
style.css, откуда нужно скопировать все и добавить в свой scss-файл(icons.scss), подключенный к styles.css и удалить этот файл, оставить только icons.ttf(в папке fonts).
далее в icons.scss изменить путь к файлу icons.tts и в [class^=]-что-то там добавить псевдожлементы ::before. теперь нужно скопировать нужный класс
и добавить его к нужному объекту(РАБОТАЕТ ДЛЯ ПРОСТЫХ ИКОНОК). 

3. Если картинка в флекс-контейнере, ширина указывается не через 'width', а через 'flex 0 0 200px'.
