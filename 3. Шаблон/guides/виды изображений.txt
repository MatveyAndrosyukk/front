Бывают растровые и векторные.
Растровые состоят из пикселей.
Векторная графика создает элементы из фигур, расчитанных по мат. формулам.
Преимущество векторной вт ом, что при увеличении масштаба качество не страдает.
JPEG - растровый формат, который отлично поддается оптимизации(подходит для контентных изображений).
PNG - растровый формат, но с фишкой, что отдельные его части могут быть прозрачными, он плохо подходит для оптимизации(подходит для фоновых изображений и элементов дизайна).
GIF - растровый формат, его фишка это анимация, позволяет хранить сжатые данные без потерь качества до 256 цветов.
SVG - векторный формат, как целый язык разметки с тегами, аттрибутами из начениями(подходит для иконок).
WEBP - замена всем растровым форматам(анимация, как gif, прозрачность, как png, минимальный вес), не поддерживается во всех браузерах.
ICO - используется для отображения иконки в браузере(размер 16х16).
